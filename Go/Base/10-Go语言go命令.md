# 10-Go语言go命令

### go命令

- `go version` - 查看版本
- `go env` - 查看环境变量
- `go fmt` - 格式化代码
- `go run` - 编译并运行
- `go build` - 编译检查
- `go install` - 安装程序


```go
package main

import (
	"fmt"
)

// go version - 查看版本
// go env - 查看环境变量
// go fmt - 格式化代码
// go run - 编译并运行
// go build - 编译检查
// go install - 安装程序
func main() {
	fmt.Println("Go..")

}

```
