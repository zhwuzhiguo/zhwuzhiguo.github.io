# 日志简介

* 二进制日志：二进制文件的形式记录数据库中的操作，但不记录查询语句。
* 错误日志：记录MySQL服务器启动、关闭和运行错误等信息。
* 通用查询日志：记录用户登录和记录查询的信息。
* 慢查询日志：记录执行时间超过指定时间的操作。

除二进制日志外，其他日志都是文本文件。  
默认情况只启动错误日志，其他日志需要管理员设置。  
启动日志会降低数据库执行速度，并会占用大量磁盘空间。
