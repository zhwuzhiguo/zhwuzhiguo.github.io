# docker 安装 nexus


## nexus2

```
sudo chown -R 200 /home/ubuntu/docker/nexus/data
sudo docker run -d -p 8081:8081 --name nexus -e TZ=Asia/Shanghai -e MIN_HEAP=256m -e MAX_HEAP=512m -v /home/ubuntu/docker/nexus/data:/sonatype-work sonatype/nexus
```

## nexus3

```
sudo chown -R 200 /home/ubuntu/docker/nexus3/data
sudo docker run -d -p 8082:8081 --name nexus3 -e TZ=Asia/Shanghai -v /home/ubuntu/docker/nexus3/data:/nexus-data sonatype/nexus3
```
